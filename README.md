# SimpsonBook

An iOS app showcasing Simpson family characters. Browse the list and tap to view details about each character.

## Features

- TableView list of Simpson characters
- Character details: name, job, image
- Includes: Homer, Marge, Bart, Lisa, Maggie, Mr. Burns
- Navigation to detail view

## Requirements

- Xcode 15+
- iOS 15.0+

## Getting Started

1. Clone and open in Xcode:
   ```bash
   git clone https://github.com/cardaozel/SimpsonBook.git
   cd SimpsonBook
   open SimpsonBook.xcodeproj
   ```

2. Run on simulator or device (⌘R)
3. Tap a character to see details

## Project Structure

- `ViewController.swift` — Character list (TableView)
- `detailsVC.swift` — Character detail screen
- `Simpson.swift` — Character model
